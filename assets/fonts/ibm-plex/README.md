# IBM Plex font subset

This directory contains the four static IBM Plex faces used by the Perform
workspace:

- IBM Plex Sans Condensed Medium
- IBM Plex Sans Condensed SemiBold
- IBM Plex Mono Medium
- IBM Plex Mono SemiBold

They were taken without modification from the official
[`IBM/plex`](https://github.com/IBM/plex) repository at commit
`2f9ba1b25957d958db71a849e85d72e3ecfb845a`.

The fonts are distributed under the SIL Open Font License 1.1. See `OFL.txt`.
